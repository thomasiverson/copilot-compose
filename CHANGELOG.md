## [0.1.0] - 2026-03-19

### Added
- **Initial Release**
- **Sync Command:** Fetch composable instruction modules from a master repo, resolve version references, compose into a single `.github/copilot-instructions.md` file, and commit to the current workspace
- **Preview Command:** Display a diff view showing exactly what would be added, changed, or removed by running sync (without applying changes)
- **Init Command:** Interactive guided setup to create `.github/copilot-compose.yaml` with questions about source repo, modules, and presets
- **Check Command:** Verify sync status—confirm that `.github/copilot-compose.yaml` exists and that the composed instructions file is current
- **Browse Command:** Explore available modules, presets, and their metadata from the master repo (read-only interface)
- **Status Bar Indicator:** Real-time sync status display (✓ in sync, ⚠ out of sync, ⏳ syncing)
- **Auto-Sync on Workspace Open:** Automatically run sync when a workspace with `.github/copilot-compose.yaml` is opened (configurable)
- **Composable Instruction Modules:** Support for Markdown files with YAML frontmatter (id, version, tags, scope, dependencies, conflicts) stored in a central master repo
- **Version Pinning:** Lock module versions to a specific branch, tag, or commit SHA via the `ref` field
- **Module Overrides:** Prepend or append project-specific content to any module (without forking the module)
- **Path-Specific Instructions:** Include different modules for specific file patterns (e.g., SQL performance tips for `**/*.sql`)
- **Preset Support:** Use predefined module collections ("team-frontend", "team-backend") instead of listing modules individually
- **Settings:**
  - `copilotCompose.masterRepoUrl` — Override source URL for all workspaces
  - `copilotCompose.autoSyncOnOpen` — Enable/disable auto-sync on workspace open (default: true)
  - `copilotCompose.autoSyncInterval` — Interval for background sync checks in seconds (default: 3600)
  - `copilotCompose.showStatusBar` — Show/hide sync status indicator (default: true)
  - `copilotCompose.composedFileHeader` — Customize the "DO NOT EDIT" header in composed files
- **Configuration File:** `.github/copilot-compose.yaml` with full support for modules, presets, refs, overrides, and path-specific instructions
- **Composed File:** `.github/copilot-instructions.md` generated by composition engine; committed to repo for visibility and code review
